<pre>  ol {
    margin: 40px auto 0;
    padding: 0;
    counter-reset: list1;
    width:778px;
        &amp; li {
          list-style:none;
          margin: 0 0 15px 0;
          padding: 0;
          text-align:center;
          font-size: 16px;
          font-weight: 500;
              &amp;:before {
                counter-increment: list1;
                content: counter(list1) ". ";
              }
          @for $ol from 1 through 18{
            &amp; ol.part-#{$ol}  {
              counter-reset: list2;
              margin:0;
                &amp; li {
                  margin:0 0 5px;
                    &amp;:before {
                      counter-increment: list2;
                      content: counter(list1) ". " counter(list2) ". ";
                    }
                }
            }
          }
          &amp; h2{
            display:inline-block;
            font-size: 16px;
            font-weight: 500;
            margin: 0 0 5px;
          }
          &amp; ol  {
            &amp; li{
              margin: 0px;
              text-align:left;
              font-size: 14px;
              font-weight: 300;
                &amp;:before{
                  list-style-type: none;
                }
            }
          }
       }
}

<p>
	Итак имеем массив текста разделённый на ненумерованные списки.
</p>
<ul>
	<li>Задаём разделам нужный класс CSS здесь указал (.part-(+порядковый номер раздела), например .part-1, part-2, part-3 и так далее)</li>
	<li>Задаём основной массив &lt;ol&gt; с установкой на него счётчика {counter-reset: list1;}</li>
	<li>Далее необходимо его запустить. Запускать его будем в &lt;li&gt;, тут уже без :before не обойтись {counter-increment: list1; content: counter(list1) ". ";}
	<ul>
		<li> {counter-increment: list1;}  - показывает, то что данный &lt;li&gt; использует дерево нумерации массива list1</li>
		<li>". " - Это формат вывода чисел в нумерации - (1.1., 1.2, 2.1, 2.2 и так далее), в данном случае с точкой в конце. </li>
		<li>с первым уровнем дерева нумерации разобрались </li>
	</ul>
	</li>
	<li>Для решения вопроса вывода последующей нумерации я создал массив  @for $ol from 1 through 18{ ... }
	<ul>
		<li>с помощью функции @for в SASS мы можем задать количество повторений</li>
		<li>придумываем название функции, в данном случае $ol</li>
		<li>задаём то количество пунктов сколько пунктов в планируемом документе в данном случае 18 пунктов договора:  @for $ol from 1 through 18. Это позволит повторять наш код для всех тегов в массиве от 1 до 18</li>
		<li>важно отметить, что необходимо добавить возможность автоматического добавления класса переменной $ol. Как это сделать: - добавить внутрь функции<br>
		@for $ol from 1 through 18{ ... }
		<br>
		&amp; ol.part-#{$ol}  {
              counter-reset: list2;} - это инициирует второй пункт документа. А также все последующие, от 1 до 18(и сколько укажите)
		</li>
		<li>теперь необходимо вывести сам текст с подпунктами внутрь родительского элемента &lt;ol&gt;: &amp; ol.part-#{$ol} { counter-reset: list2;}<br>
		&amp; li {
                  margin:0 0 5px;
                    &amp;:before {
                      counter-increment: list2;
                      content: counter(list1) ". " counter(list2) ". ";
                    }
                } . Здесь мы указываем, что content: вводит на родительский номер и номер подпункта, через точки разделители(ну или точки на конце)
		</li>
	</ul>
	</li>
	<li>Замечу, что постоянные атрибуты вынесены вне повторяющихся массивов, чтобы не плодить повторяющийся код.</li>
